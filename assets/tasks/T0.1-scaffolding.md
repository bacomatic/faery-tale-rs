# T0.1 — Repo scaffolding & shared helpers

> Read [`_SHARED.md`](_SHARED.md) first. Two-agent task: Implementer + separate Verifier.

## Dependencies
None. (Unblocks all of Wave 1 & 2.)

## Inputs (read-only)
- `../plan.md` §"Output layout (committed `assets/`)"

## Implementation
1. Create `assets/` subdirs: `palettes/ tiles/ sprites/ masks/ world/ audio/{music,instruments,sfx}/ fonts/ screens/ shaders/ tables/ text/ formats/`.
2. Add a `game/` symlink → `../faery-tale-rs/game` (or document the `--image` alternative in the tool's `--help`).
3. Create `tools/asset_common.py` providing:
   - arg helpers for `--game-dir` (default `../faery-tale-rs/game`) and `--src-dir` (default `src/`);
   - `write_indexed_png(path, idx_array, palette)` — palette indices 0–31, exact;
   - `write_rgba_png(path, rgba_array)`;
   - `rgb4_to_rgba8(v)` — convert `rgb4` (Amiga OCS 12-bit `0x0RGB`) by nibble-replication (`0xF→0xFF`);
   - `write_highlight_mask(path, idx_array)` — 1 bit/pixel where index ∈ 16–24, transparency follows index 31;
   - `write_json(path, obj)` — stable key order, deterministic bytes.

## Outputs
- `assets/` dir tree, `game/` symlink, `tools/asset_common.py`, `tools/tests/test_asset_common.py`.

## Implementer self-check
- `rgb4_to_rgba8(0x0F0F)` etc. correct; indexed PNG round-trips; mask bits correct.
- `pytest tools/tests/test_asset_common.py` passes.

## Verification (DIFFERENT agent)
Do not reuse the Implementer's tests. Independently:
1. List `assets/` and confirm every subdir from the plan exists; confirm `game/` resolves to the real game dir.
2. Write a throwaway script that calls each `asset_common` helper with hand-computed inputs and asserts outputs by hand (e.g. `0x0FFF → (255,255,255,255)`, `0x0123 → (0x11,0x22,0x33,255)`).
3. Round-trip: build a small index array containing values {0, 15, 16, 24, 25, 31}, write indexed PNG, re-read with an independent PNG reader (e.g. Pillow), assert identical indices.
4. Build the same array, write a highlight mask, assert bits set **only** for 16..24 and that index 31 maps to transparent.
5. Re-run the Implementer's pytest from a clean state; confirm green.
Report PASS/FAIL with the concrete values you checked.
