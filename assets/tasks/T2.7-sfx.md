# T2.7 — SFX extractor

> Read [`_SHARED.md`](_SHARED.md) first. Two-agent task: Implementer + separate Verifier.

## Dependencies
T0.1.

## Inputs (read-only)
- ADF `game/image` blocks 920–930 (6 PCM SFX samples)
- `audio.rs` (Rust oracle, if it decodes SFX)

## Implementation
Create `tools/extract_sfx.py`:
1. Extract 6 samples from ADF blocks 920–930, 8-bit PCM.
2. Write WAV. Document the original Paula playback rate ≈ 8000 Hz. **Do NOT resample** — store native PCM.

## Outputs
- `assets/audio/sfx/sfx_<n>.wav` ×6
- `tools/tests/test_extract_sfx.py`

## Implementer self-check
- WAV PCM bytes match the native samples; rate documented; pytest on block offsets.

## Verification (DIFFERENT agent)
1. For 2 of the 6 SFX, read the raw block bytes from the ADF yourself and compare against the
   WAV data chunk — must be byte-identical (no resampling, no normalization).
2. Confirm exactly 6 WAV files; check WAV header sample rate matches the documented native rate.
3. Confirm 8-bit PCM (not 16-bit upconvert) unless the spec explicitly documents a lossless container choice.
4. Spot-play one WAV (or inspect waveform) for sanity.
Report PASS/FAIL with the SFX checked.
