# T1.1 — Palettes extractor

> Read [`_SHARED.md`](_SHARED.md) first. Two-agent task: Implementer + separate Verifier.

## Dependencies
T0.1 (`asset_common`), T0.2 (C-array extractor).

## Inputs (read-only)
- `src/fmain.c` (`pagecolors`, `textcolors`, `introcolors`, `sun_colors`, `blackcolors`)
- `src/fmain2.c` (`fade_page` — region color-31 overrides)

## Implementation
1. Create `tools/extract_palettes.py`.
2. Extract `pagecolors`, `textcolors`, `introcolors`, `sun_colors`, `blackcolors` from `fmain.c`.
3. Per-region color-31 overrides from `fade_page` in `fmain2.c`: region 4 = `0x0980`,
   region 9 = `0x0445`, all others = `0x0bdf`.
4. Emit each entry as `{index, rgb4 (0x0RGB Amiga OCS 12-bit), rgba8}` using `asset_common.rgb4_to_rgba8`.

## Outputs
- `assets/palettes/{pagecolors,textcolors,introcolors,sun_colors}.json`
- `assets/palettes/region_overrides.json`
- `tools/tests/test_extract_palettes.py`

## Implementer self-check
- Color counts per palette match source; `rgb4`↔`rgba8` consistent; pytest green.

## Verification (DIFFERENT agent)
Do not trust the Implementer's conversion. Independently:
1. Hand-locate `pagecolors[]` in `fmain.c`; transcribe entries 0, 16, 24, 31 and convert
   each `0x0RGB → rgba8` by hand; diff against the JSON.
2. Cross-check `pagecolors[31]` == `0x0bdf` against memory `region_palette` note, and confirm
   `region_overrides.json` has region 4 = `0x0980`, region 9 = `0x0445`, default `0x0bdf`.
3. **Rust oracle:** dump `palette.rs` `amiga_color_to_rgba()` for a few values, diff vs JSON `rgba8`.
4. Re-run pytest clean.
Report PASS/FAIL with the converted values.
