# T2.4 — IFF/ILBM screens extractor

> Read [`_SHARED.md`](_SHARED.md) first. Two-agent task: Implementer + separate Verifier.

## Dependencies
T0.1.

## Inputs (read-only)
- `game/` IFF files: `p1a`, `p1b`, `p2a`, `hiscreen`, … (intro/placard/hi-score)
- `iff_image.rs` (Rust oracle)

## Implementation
Create `tools/extract_screens.py`:
1. Parse IFF/ILBM chunks: `BMHD`, `CMAP`, `BODY` with **ByteRun1** decompression.
2. Convert to RGBA PNG. Mirror `iff_image.rs` logic.

## Outputs
- `assets/screens/<name>.png`
- `tools/tests/test_extract_screens.py`

## Implementer self-check
- Each screen decodes; RGBA diff-clean vs `iff_image.rs`; pytest covers ByteRun1.

## Verification (DIFFERENT agent)
1. **Rust oracle:** dump `iff_image.rs` output for at least 2 screens and pixel-diff vs the PNGs — zero diff.
2. **ByteRun1:** craft a tiny known compressed run and decode it with the Implementer's routine; assert expected bytes.
3. Confirm BMHD dimensions match the PNG dimensions for each screen.
4. Spot-open one screen and sanity-check it visually (not garbage).
5. Re-run pytest clean.
Report PASS/FAIL with the screens diffed.
