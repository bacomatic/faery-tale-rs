# T2.3 — Shadow/collision masks extractor

> Read [`_SHARED.md`](_SHARED.md) first. Two-agent task: Implementer + separate Verifier.

## Dependencies
T0.1.

## Inputs (read-only)
- ADF `game/image` (mask table)

## Implementation
Create `tools/extract_masks.py`:
1. 256 entries × 64 B (32 rows × 2 B, 1 bit/pixel) from the ADF.
2. Emit indexed PNG per mask + a JSON describing the bit layout (row stride, bit order, dims).

## Outputs
- `assets/masks/*.png` + bit-layout JSON
- `tools/tests/test_extract_masks.py`

## Implementer self-check
- Bit layout round-trips (PNG→bits == source bytes); 256 entries; pytest green.

## Verification (DIFFERENT agent)
1. Pick entry 0 and entry 255: read the raw 64 bytes from the ADF yourself and reconstruct the
   1-bit bitmap by hand; pixel-diff against the emitted PNG.
2. Confirm exactly 256 mask PNGs and that each is 16×32 (or as documented), 1 bit/pixel.
3. Validate the JSON bit-layout description by round-tripping PNG→bits and comparing to source bytes.
4. Re-run pytest clean.
Report PASS/FAIL with the two entries checked.
