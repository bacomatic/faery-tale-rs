# T1.4 — Narrative text extractor

> Read [`_SHARED.md`](_SHARED.md) first. Two-agent task: Implementer + separate Verifier.

## Dependencies
T0.1.

## Inputs (read-only)
- `../faery-tale-rs/faery.toml` `[narr]`
- `reference/.../narr-asm-*.md` (cross-check)

## Implementation
1. Create `tools/extract_text.py`. Pull `[narr]` strings from `faery.toml`.
2. Split into `event_msg.json`, `speeches.json`, `place_msg.json`, `inside_msg.json`.
3. Preserve `%` (player name) and `$` (target) placeholders verbatim.

## Outputs
- `assets/text/{event_msg,speeches,place_msg,inside_msg}.json`
- `tools/tests/test_extract_text.py`

## Implementer self-check
- Placeholder chars preserved; string counts match `[narr]`; pytest green.

## Verification (DIFFERENT agent)
1. Independently count `[narr]` entries in `faery.toml` and compare to the sum across the 4 JSON files.
2. Grep the JSON for `%` and `$` and confirm they survive (pick 3 strings known to contain them).
3. Cross-check 2–3 strings against `reference/.../narr-asm-*.md`.
4. Confirm no encoding mangling (non-ASCII, escapes) vs source.
Report PASS/FAIL with the counts.
