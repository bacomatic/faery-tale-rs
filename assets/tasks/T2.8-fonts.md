# T2.8 — Fonts extractor

> Read [`_SHARED.md`](_SHARED.md) first. Two-agent task: Implementer + separate Verifier.

## Dependencies
T0.1.

## Inputs (read-only)
- `game/fonts/` (`.font` + DiskFont data, ID `0x0F80`)
- `font.rs` (Rust oracle)

## Implementation
Create `tools/extract_fonts.py`:
1. Parse `.font` + DiskFont (ID `0x0F80`): glyph bitmaps.
2. Emit per-glyph PNG + a packed glyph atlas.
3. Emit metrics JSON: `y_size`, `baseline`, `modulo`, `lo_char`/`hi_char`, `char_loc` (offset+width).
   Mirror `font.rs`.

## Outputs
- `assets/fonts/<font>/glyphs/*.png`, `assets/fonts/<font>/<font>.json`
- `tools/tests/test_extract_fonts.py`

## Implementer self-check
- Glyph bitmaps diff-clean vs `font.rs`; metrics correct; pytest green.

## Verification (DIFFERENT agent)
1. **Rust oracle:** dump `font.rs` glyph bitmaps and metrics; pixel-diff several glyphs (incl.
   first and last in `lo_char`..`hi_char`) vs the PNGs — zero diff.
2. Confirm metrics JSON fields match the DiskFont header values you read yourself.
3. Confirm `char_loc` offsets+widths correctly index into the packed atlas (no overlap/overflow).
4. Re-run pytest clean.
Report PASS/FAIL with the glyphs diffed.
