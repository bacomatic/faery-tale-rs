# T2.1 — Sprites extractor extension

> Read [`_SHARED.md`](_SHARED.md) first. Two-agent task: Implementer + separate Verifier.

## Dependencies
T0.1.

## Inputs (read-only)
- `tools/extract_sprites.py` (already decodes all 18 cfiles: 5 bitplanes, transparency = index 31, `pagecolors[]`)
- `CFILES` table (frame counts) in the source
- Existing `sprite_output/` (730 files) for regression diff

## Implementation
Extend `tools/extract_sprites.py`:
1. Per-actor JSON atlas: frame→rect, dimensions, transparency index 31, frame counts from `CFILES`.
2. Add an **indexed-PNG** output mode (current output is RGBA).
3. Emit into `assets/sprites/<actor>/`: `frame_000.png …`, `<actor>_sheet.png`, `<actor>.json`.
4. Emit a **1-bit highlight mask** per sheet: `<actor>_highlightmask.png` (index ∈ 16–24; transparency follows 31).
5. Keep labeled/2x debug variants OUT of the shipped bundle.

## Outputs
- `assets/sprites/<actor>/{frame_*.png, <actor>_sheet.png, <actor>_highlightmask.png, <actor>.json}` ×18
- `tools/tests/test_extract_sprites.py`

## Implementer self-check
- RGBA frames diff-clean vs `sprite_output/`; indexed PNG round-trips; mask matches 16–24; pytest green.

## Verification (DIFFERENT agent)
1. **Regression:** byte/pixel-diff new RGBA frames against existing `sprite_output/` — zero diff.
2. **Round-trip:** re-decode an indexed sprite PNG to indices and compare against the raw
   bitplane decode for the same actor; assert RGBA == `pagecolors`-applied indices.
3. **Mask correctness:** for one hero sheet (`julian`/`phillip`/`kevin`, uses index 24) and one
   NPC/enemy sheet, confirm mask bits are set exactly where source index ∈ 16–24.
4. **Atlas:** confirm `<actor>.json` frame count == `CFILES` and every rect lands in-bounds.
5. Re-run pytest clean.
Report PASS/FAIL with the diff result and the actors checked.
